# 05. Coding Agent 只读工具

## 本步目标

在 `@kairos/coding-agent` 里加入最小代码阅读能力。

## 为什么放在 coding-agent

| 放置位置 | 判断 |
| --- | --- |
| `@kairos/ai` | 不合适。模型层不应该知道文件系统 |
| `@kairos/agent` | 不合适。通用 Runtime 不应该绑定代码任务 |
| `@kairos/coding-agent` | 合适。代码工具和工作区边界属于这里 |

## 当前只读工具

| 工具 | 作用 |
| --- | --- |
| `read_file` | 读取工作区内文件 |
| `list_dir` | 查看目录内容 |
| `grep` | 搜索文本或正则 |

## 安全边界

只读工具默认限制在指定工作区内，不允许通过 `../` 跳出工作区。

## 学到什么

Agent 做代码任务时，第一个能力应该是“看清楚文件”。在没读过相关文件前就编辑，风险很高。
