# 术语表

| 术语 | 含义 |
| --- | --- |
| Provider | 模型服务来源，例如 OpenCode Go |
| Model | 某个 provider 下的具体模型 |
| Runtime | Agent 的运行时，负责消息、工具和循环控制 |
| Tool | 本地可执行能力，例如读文件、搜索、编辑文件 |
| Tool Call | 模型发出的工具调用请求 |
| Tool Result | 本地工具执行后的结果 |
| Middleware | 工具执行前后的扩展机制 |
| TUI | 命令行界面 |
| Todo | Agent 自己维护的任务状态 |
| Workspace | Agent 被允许操作的项目目录 |
| Workspace Diff | 当前工作区相对 git 的改动 |
